git status --> to check status of files

git add <files> --> to stage files
or
git add . --> to stage all files

git rm --cached <file> --> to unstage

git commit -m <comment> --> to commit files locally

git remote add origin <repo link> --> to connect with github repo (req one time only)

git branch -M main --> add to main branch

git push -u origin main --> push to main branch in github
